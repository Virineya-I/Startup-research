# -Startup-research
Финансовая компания, работающая с венчурными инвестициями, хочет понять закономерности финансирования стартапов и оценить перспективы выхода на рынок с покупкой и развитием компаний. Цель проекта — подготовить датасет к работе, исследовать динамику и структуру финансирования стартапов и ответить на вопросы для оценки инвестиционных стратегий.
